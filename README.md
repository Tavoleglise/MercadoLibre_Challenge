# Mercado Libre Challenge

This project was a Tech Frontend challenge for Mercado Libre

## How to install and run

In the server folder, run:

### `npm install`

### `npm run start_server`

Available endpoints:

Port Used : 3002 <br />
/api/items?search= <br />
/api/items/:id <br />

In the main directory, run:

### `npm install`

### `npm run start`
